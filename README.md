校园活动报名系统说明

校园活动报名系统旨在规范校园内各类活动的报名流程。学校拟开发该系统，并将其划分为以下子系统：

1. 用户管理子系统

“用户”指的是此软件系统的使用者。
该子系统主要负责处理与系统用户相关的功能，包括但不限于：

用户注册：新用户创建账号

用户登录：注册用户通过凭证进入系统

用户信息管理：查看与修改个人信息

2. 活动管理子系统

该子系统主要负责校园活动信息管理，包括但不限于：

活动发布：发布新的校园活动

活动更新：修改或维护活动信息

活动查询：查看活动列表及详情

3. 报名管理子系统

该子系统主要负责活动报名流程的管理，包括但不限于：

活动报名：用户报名参加活动

报名审核：管理员审核报名信息

报名通知：向用户发送审核结果或相关通知
